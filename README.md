1-->
var -- function scoped can be redeclared and reassigned and hoisted with undefined.
let -- block-scoped can be reassigned but not redeclared in the same scope, hoisted but uninitialized.
const -- block-scoped cannot be reassigned or redeclared, hoisted but uninitialized for objects/arrays reference is constant but content can change.

2-->
The spread operator expands an array, object, or iterable into individual elements, which is often used to copy, merge, or pass multiple values.

3-->
map --> creates a new array by transforming each element.
filter --> creates a new array with elements that pass a condition
forEach --> executes a function on each element but does not return a new array.

4-->
Arrow function --> An arrow function is a shorthand syntax for writing functions in JavaScript.

5--> 
Template literals are JavaScript string literals that are enclosed in backticks allowing embedded expressions and supporting multi-line strings.
